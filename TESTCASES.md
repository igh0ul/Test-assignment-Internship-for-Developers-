# Тест-кейсы и автоматизированные Е-2-Е тесты
***
## тест-кейсы для сайта avito vercel
***
#### TC-01: Создание задачи с валидными данными
**Предварительные условия:** Пользователь авторизован, открыт список задач  
**Шаги:**
1. Нажать кнопку "Создать задачу"
2. Заполнить поля: Название, описание, проект, приоритет, статус, исполнитель
3. Нажать "Создать"
**Ожидаемый результат:**
- Созданная вами задача появляется в списке с корректным заголовком и остальными данными


---

#### TC-02: Ошибка при пустом поле (любом)
**Предварительные условия:** Пользователь авторизован, открыт список задач 
**Шаги:**
1. Нажать "Создать задачу"
2. Оставить поле любое одно пустым
3. Нажать "Сохранить"
**Ожидаемый:**
- Задача не создаётся

---


### 2. Открытие карточки задачи

#### TC-03: Просмотр созданной задачи
**Предварительные условия:** Есть задача в списке  
**Шаги:**
1. Кликнуть по задаче
**Ожидаемый результат:**
- Открывается карточка задачи и ее данные корректно отображаются




---

### 3. Поиск задачи

#### TC-04: Поиск по заголовку
**Предварительные условия:** Пользователь авторизован, открыт список задач, есть задача в списке 
**Шаги:**
1. Ввести существующий заголовок задачи в поиск
2. Нажать Enter
**Ожидаемый результат:**
- В результатах отображается нужная задача

---

#### TC-05: Поиск по статусу   
**Предварительные условия:** Пользователь авторизован, открыт список задач, есть задача в списке
**Шаги:**
1. кликнуть на плейсхолдер "статус"
2. выбрать статус задачи 
**Ожидаемый результат:**
- Найдены все задачи с выбранным статусом

---

#### TC-06: Поиск по несуществующему запросу
**Шаги:**
1. Ввести случайную строку
**Ожидаемый результат:**
- Возвращается состояние: “Задачи не найдены”

---

#### TC-05: Поиск по доске   
**Предварительные условия:** Пользователь авторизован, открыт список задач, есть задача в списке
**Шаги:**
1. кликнуть на плейсхолдер "доска"
2. выбрать доску задачи 
**Ожидаемый результат:**
- Найдены все задачи из выбранной доски

---

### 4. Переход на доску проекта

### TC-06: Переход на Kanban-доску
**Шаги:**
1. Нажать кнопку проекты
2. Нажать на кнопку "перейти к доске"
**Ожидаемый результат:**
- Открывается Kanban-доска с колонками: Backlog, In Progress, Done

---

#### TC-07: Карточка задачи отображается на доске
**Шаги:**
1. Создать задачу
2. Открыть доску
**Ожидаемый результат:**
- Задача появляется в колонке Backlog

---

#### TC-08: Перемещение задачи по колонкам
**Шаги:**
1. Нажать кнопку проекты
2. Нажать на кнопку "перейти к доске"
3. Нажать на любой проект
4. нажать на поле статус и выбрать другой статус задачи
**Ожидаемый результат:**
- На карточке статус обновляется и задача перемещается в сооответствующую статусу колонку
